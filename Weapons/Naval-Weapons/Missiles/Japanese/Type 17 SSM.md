# Type 17 SSM (SSM-2)

## Overview
**Nation**: Japanese
**Type**: SSM (Anti-ship)
**Year**: 2018

## Specifications
- **Diameter**: 13.8"
- **Length**: 16.4 ft
- **Warhead**: 496 lbs
- **Speed**: Mach 0.90
- **Range**: 216.0 nm

## System
- **Propulsion**: Solid rocket booster + Mitsubishi TJM2 turbojet
- **Guidance**: INS/GPS + AESA radar terminal homing
- **Target**: Surface ships, land targets
- **Platform**: Maya-class destroyers (Mk-141 quad launchers)

## Notes
Ship-launched derivative of Type 12 SSM. Doubled range over Type 90 predecessor. Features Ka-band AESA terminal seeker. Post-launch target update capability. Simultaneous multi-missile coordination. Land-attack capable via GPS coordinates. First deployed on Maya-class (2020).
