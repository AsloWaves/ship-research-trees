## German Naval Guns

### Battleship Main Battery

#### 28cm/45 SK L/45 Pattern 1907
- **Designation**: 11"/45 (28cm) SK L/45
- **Caliber**: 11.0 inches (280mm)
- **Ships**: Nassau-class battleships (Nassau, Westfalen, Rheinland, Posen), Von der Tann battlecruiser
- **Barrel Length**: 45 calibers
- **Year**: 1907
- **Note**: First dreadnought guns for German Navy; actual bore 28.3cm (11.1")

#### 30.5cm/50 SK L/50 Pattern 1909
