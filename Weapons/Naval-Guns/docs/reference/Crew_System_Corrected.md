# WOS2.3 Crew Classification System - Navy Field Style (CORRECTED)

## üìã Core Philosophy

**CRITICAL**: Crew LEVELS are completely independent from Ship TIERS

- A **Level 100 Gunner** can serve on a **T1 Destroyer** (overperforming, expensive)
- A **Level 10 Gunner** can serve on a **T8 Battleship** (underperforming, cheap)
- Players mix crew levels based on budget, ship tier, and risk tolerance

**Navy Field System**:
- Crew has **LEVELS** (1-200+), NOT "tiers"
- At level milestones, crew can **class up** to better versions
- Each class up requires **level threshold + credits**
- Crew performance based on level vs. module level (efficiency curve)

---

## üéØ Classification Progression Structure

**All sailor types follow this pattern**:

| Classification | Level Req | Unlock Cost | Cumulative Cost | Performance vs Base |
|----------------|-----------|-------------|-----------------|---------------------|
| **[Base Name]** | Level 1 | 500‚Ç° (recruit) | 500‚Ç° | 100% |
| **Veteran [Name]** | Level 30 | 50,000‚Ç° | 50,500‚Ç° | +15% |
| **Expert [Name]** | Level 70 | 250,000‚Ç° | 300,500‚Ç° | +35% |
| **Master [Name]** | Level 120 | 1,000,000‚Ç° | 1,300,500‚Ç° | +60% |
| **Chief [Name]** | Level 180 | 3,500,000‚Ç° | 4,800,500‚Ç° | +100% |

**How It Works**:
1. Recruit Level 1 Light Gunner (500‚Ç°)
2. Crew gains XP through combat, reaches Level 30
3. Pay 50,000‚Ç° to upgrade to "Veteran Light Gunner"
4. Crew continues leveling, reaches 70, 120, 180...
5. At each milestone, pay to unlock next classification

## üìä Level-Based Efficiency System

### **Crew Level vs. Module Level Matching**

**Perfect Match** (Crew Level = Module Level):
- **100% efficiency** (baseline performance)

**Crew Underleveled** (Crew Level < Module Level):
- **Efficiency penalty** based on level gap
- Example: Level 50 crew on Level 100 turret = 75% efficiency

**Crew Overleveled** (Crew Level > Module Level):
- **Efficiency bonus** based on level gap (capped)
- Example: Level 150 crew on Level 80 turret = 125% efficiency (capped at 150%)

### **Efficiency Curve Examples**

**Level 90 Turret Performance**:
```
Level 1 crew   = 35% efficiency (huge penalty, -65%)
Level 30 crew  = 55% efficiency (moderate penalty, -45%)
Level 45 crew  = 65% efficiency (half level, -35%)
Level 70 crew  = 85% efficiency (close but not optimal, -15%)
Level 90 crew  = 100% efficiency (perfect match)
Level 120 crew = 115% efficiency (overperforming, +15%)
Level 180 crew = 130% efficiency (extreme overperformance, +30%, capped)
```

**Real Scenario Examples**:

**Scenario 1**: *Captain Murphy's Budget Build*

- **Result**: Functional but underperforming, affordable

- **Turret Efficiency**: 125% (capped bonus for 40-level advantage)

## üéØ Strategic Implications

### **Early Game** (T1-T3)
- Use Level 1 base crew (500‚Ç° each), cheap and disposable
- Upgrade to Veteran (Level 30) as budget allows
- Focus on 3-4 core positions (gunner, engineer, DC, bridge)
- **Philosophy**: Minimize crew investment, maximize learning

### **Mid Game** (T4-T6)
- Upgrade to Expert (Level 70) for critical positions
- Mix of Veteran/Expert based on importance
- Start specializing (different gunner types, torpedo vs. AA focus)
- **Philosophy**: Balanced investment, competitive performance

### **Late Game** (T7-T9)
- Master (Level 120) for main combat positions
- Expert for support positions
- Consider backup crews for permadeath zones
- **Philosophy**: Performance focus, risk management

### **Endgame** (T10)
- Chief (Level 180) only for the wealthiest
- Most players use Master/Expert mix on T9 ships instead
- T10 with Chief crew = 100M‚Ç°+ investment with 100% permadeath
- **Philosophy**: Prestige vs. practicality
