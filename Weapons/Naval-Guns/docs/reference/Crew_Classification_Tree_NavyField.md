# WOS2.3 Crew Classification Tree - Navy Field Style

## üìã Neutral Sailor System

**All crew starts as Neutral Sailors** at Level 1, then branches into 42 different classification paths at various level milestones.

---

## üåü NEUTRAL SAILOR (Universal Base)

**Level Range**: 1-19
**Cost**: 500‚Ç° (initial recruitment)
**Performance**: Base performance on all positions (100% baseline)

**At Level 20+**: Choose ONE classification path permanently

**Available Paths**: 42 total classifications across 8 divisions:
- 4 Gunnery types
- 2 Torpedo types
- 1 AA type
- 5 Engineering types
- 12 Aviation types
- 8 Submarine types
- 4 ASW types
- 3 Command types

**Important Rules**:
- ‚ùå **No respeccing** - Classification choice is PERMANENT
- ‚úÖ **Free transfer** - Crew cards transfer between ships freely
- ‚úÖ **Universal base** - Level 1-19 Neutral can work any position (poorly)
- ‚ö†Ô∏è **Efficiency penalty** - Unclassified crew on specialized positions = 50% efficiency penalty

---

## üéØ Classification Progression Philosophy

**Progression Speed Categories**:

**Fast Track** (Early specialists, common roles):
- First classification: Level 20
- Quick advancement through ranks
- Total progression: ~170 levels to max
- **Examples**: Light Gunner, Torpedo Man, AA Gunner, Engineer

**Standard Track** (Balanced progression):
- First classification: Level 25
- Moderate advancement pace
- Total progression: ~180 levels to max
- **Examples**: Medium Gunner, Heavy Gunner, most support roles

**Advanced Track** (Late bloomers, complex roles):
- First classification: Level 30
- Slower advancement, higher requirements
- Total progression: ~190 levels to max
- **Examples**: Super Heavy Gunner, Aviation specialists, Submarine roles

**Elite Track** (Prestige positions, command roles):
- First classification: Level 35
- Very slow advancement, maximum prestige
- Total progression: ~200 levels to max
- **Examples**: Command Division, specialized aviation, advanced submarine
