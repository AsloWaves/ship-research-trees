# Historical Naval Gun Turret Research - Big 4 Nations

Comprehensive research on historical naval gun turrets for USA, Britain, Germany, and Japan.

## United States Navy Turrets

### Iowa Class - Mark 7 16"/50 Triple Turret
- **Designation**: Mark 7 Triple Turret
- **Gun**: 16"/50 Mark 7
- **Guns per Turret**: 3
- **Weight**: ~2,500 tons (estimated complete assembly)
- **Elevation**: -5° to +45°
- **Elevation Speed**: Up to 12°/second
- **Traverse Speed**: ~4°/second
- **Traverse Arc**: ~300°
- **Ships**: Iowa class (Iowa, New Jersey, Missouri, Wisconsin), Montana class (planned)
- **Gun Weight**: 267,900 lbs (121.5 tons) each with breech
- **Notes**: Extended 4 decks (Turrets 1 & 3) or 5 decks (Turret 2), saved 850 tons vs 16"/45 turret

### North Carolina/South Dakota - Mark 6 16"/45 Triple Turret
- **Designation**: Mark 6 Triple Turret
- **Gun**: 16"/45 Mark 6
- **Guns per Turret**: 3
- **Weight**: 1,403-1,437 long tons (1,426-1,460 metric tons)
- **Elevation**: -5° to +45°
- **Traverse Speed**: 4°/second
- **Traverse Arc**: 150° on either side (300° total)
- **Ships**: North Carolina class, South Dakota class
- **Gun Weight**: 85.85 long tons (87.2 metric tons) each without breech
- **Crew**: 2 officers, 170 enlisted per turret
- **Barbette**: 37 feet 3 inches inside diameter
- **Gun Spacing**: 117 inches (297 cm) apart
- **Notes**: Weighed 400 tons less than Mark 2 50-caliber turret

### Colorado Class - 16"/45 Mark 1/5 Twin Turret
- **Gun**: 16"/45 Mark 1, 5, 8
- **Guns per Turret**: 2
- **Ships**: Colorado class
- **Notes**: Earlier 16" twin turret design

### New Mexico/Tennessee - 14"/50 Triple Turret
- **Gun**: 14"/50 Mark 7, 11, Mark B
- **Guns per Turret**: 3
- **Ships**: New Mexico class, Tennessee class
- **Notes**: First US triple turret design for 14" guns

### Pennsylvania Class - 14"/45 Triple Turret
- **Gun**: 14"/45 Marks 1, 2, 3, 5, 8, 9, 10, 12
- **Guns per Turret**: 3
- **Ships**: Pennsylvania class, Nevada class, New York class
- **Notes**: Early triple turret design

## Royal Navy (Britain) Turrets

### King George V Class - 14" Mark VII Quadruple Turret
- **Designation**: Mark III Quadruple Turret
- **Gun**: 14"/45 Mark VII
- **Guns per Turret**: 4
- **Weight**: 1,582 tons (quadruple)
- **Elevation Speed**: 8°/second
- **Traverse Speed**: 2°/second
- **Gun Spacing**: 96 inches (244 cm) apart
- **Ships**: King George V class (5 ships)
- **Notes**: Complex hydraulic design, initially had reliability issues later resolved

### King George V Class - 14" Mark VII Twin Turret
- **Designation**: Mark III Twin Turret
- **Gun**: 14"/45 Mark VII
- **Guns per Turret**: 2
- **Weight**: 915 tons
- **Elevation Speed**: 8°/second
- **Traverse Speed**: 2°/second
- **Gun Spacing**: 96 inches (244 cm) apart
- **Ships**: King George V class (B turret only)
- **Notes**: Hydraulic elevation by cylinder and piston, compressed air run-out

### Nelson Class - 16" Mark I Triple Turret
- **Designation**: Mark I Triple Turret
- **Gun**: 16"/45 Mark I
- **Guns per Turret**: 3
- **Weight**: Unknown (estimated >1,500 tons)
- **Turret Face Armor**: 16 inches (410 mm)
- **Side Armor**: 11 inches (280 mm)
- **Roof Armor**: 7.25 inches (184 mm)
- **Rear Armor**: 9 inches (230 mm)
- **Barbette Armor**: 15 inches (380 mm)
- **Ships**: Nelson, Rodney
- **Notes**: All turrets forward, spring-loaded vertical roller bearings due to heavy weight

### Queen Elizabeth Class - 15" Mark I Twin Turret
- **Gun**: 15"/42 Mark I
- **Guns per Turret**: 2
- **Ships**: Queen Elizabeth class, Royal Sovereign class, Vanguard
- **Notes**: Legendary 15" gun, served through both World Wars

## Imperial Japanese Navy Turrets

### Yamato Class - Type 94 46cm Triple Turret
- **Designation**: Type 94 Triple Mounting
- **Gun**: 46cm/45 Type 94
- **Guns per Turret**: 3
- **Weight**: 2,730-2,774 tonnes (complete mount)
- **Gun Weight**: 147.3 tonnes each (162.4 short tons)
- **Gun Length**: 21.13 meters (69.3 ft)
- **Elevation**: -5° to +45°
- **Elevation Speed**: 8°/second
- **Traverse Speed**: 2°/second
- **Traverse Arc**: +120° to -120° (240° total)
- **Face Armor**: 25.6 inches (650 mm)
- **Ships**: Yamato, Musashi, Shinano (converted to carrier)
- **Power**: Two independent 500 hp vertical swashplate motors
- **Notes**: Heaviest naval turrets ever built, "weighed more than a destroyer"

### Nagato Class - Type 88/Type 3 41cm Twin Turret
