# Utilities Scripts

The Utilities folder contains 1 helper script.

## Debug Utilities

- **FrustumWarningSupressor.cs** (2KB) - Suppresses Unity frustum culling warnings for 2D ocean tiles

## Purpose

This utility script filters console spam from Unity's frustum culling system when working with large 2D ocean tiles that extend beyond normal camera bounds. It's a quality-of-life improvement for development.
