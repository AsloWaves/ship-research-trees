# Leander-Class Protected Cruisers (1882)

## Overview
The Leander-class were improved protected cruisers featuring better speed and armament than earlier designs. Four ships served globally, with emphasis on colonial station duties. These ships represented incremental improvements in the protected cruiser concept, with enhanced speed and more uniform armament.

## Design

**Improved Protected Cruiser:**
- Enhanced speed over earlier classes
- Standardized 6-inch main armament
- Improved deck armor
- Better seakeeping for global operations

**Key Features:**
- Uniform 6-inch battery
- Extended range for colonial service
- Steel hull construction
- Protected machinery spaces
