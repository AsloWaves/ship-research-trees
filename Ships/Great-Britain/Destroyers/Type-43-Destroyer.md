# Type 43 Destroyer (Cancelled 1981)

## Overview
The Type 43 was a planned class of large air-defense destroyers designed 1978-1981 to replace the County-class guided missile destroyers. Featuring advanced Aegis-like phased array radar, comprehensive missile systems, nuclear propulsion options, and ASW capabilities, the Type 43 represented Britain's last attempt at building large, multi-role fleet destroyers. The class was cancelled in 1981 Defence Review due to costs (£500+ million per ship) and the decision to build more Type 42 destroyers instead. The cancellation marked Britain's shift from large, capable ships to smaller, more numerous vessels.

## Design Concept

### Mission Requirements

**Primary Roles:**
- **Air Defense**: Long-range area defense for carrier task groups
- **Anti-Surface**: Harpoon missile capability against surface threats
- **ASW**: Comprehensive anti-submarine warfare capabilities
- **Command**: Task group flagship with advanced C3 systems

**Design Philosophy:**
- One-stop replacement for County and Type 42 classes
- Aegis-like capability for Royal Navy
- Self-sufficient task group escort
- 30-year service life (1990s-2020s)
- Nuclear option for unlimited range

### Advanced Systems

**MESAR Phased Array Radar:**
- Multi-function Electronically Scanned Array Radar
- 4 fixed faces (360° coverage)
- Track-while-scan capability
- 200+ simultaneous tracks
- British answer to American SPY-1 Aegis radar

**Sea Dart Mk 2 System:**
- Improved Sea Dart with semi-active radar homing
- 88 missiles (vs 22 on Type 42)
- Extended range (80+ miles)
- Better low-altitude performance
- Twin launchers for rapid engagement

**Comprehensive Defensive Systems:**
- Sea Wolf point defense (close-in protection)
- Goalkeeper CIWS (last-ditch defense)
- Electronic warfare and decoys
- Redundant systems for survivability

**ASW Capabilities:**
- Advanced sonar suite (bow and towed array)
- ASW torpedoes
- 2× Lynx helicopters with dipping sonar and torpedoes
- Integrated ASW command system

### Propulsion Options

**Nuclear Option:**
- 2× pressurized water reactors
- Unlimited range and endurance
- Reduced crew (automation)
- Estimated cost: £600+ million per ship
- Political and cost challenges

**COGAG Option (Chosen):**
- Combined Gas turbine And Gas turbine
- 2× Rolls-Royce Olympus (boost)
- 2× Tyne (cruise)
- 5,000nm range at 18 knots
- Conventional but capable

## Cancellation (1981)

### 1981 Defence Review (Nott Review)

**Economic Context:**
- UK economic crisis and recession
- Defense budget under extreme pressure
- £200 million immediate savings required
- Choose between carriers, destroyers, or frigates

**Strategic Reassessment:**
- NATO European focus (not global power projection)
- Invincible-class carriers prioritized
- Type 42 "good enough" for North Atlantic
- Type 43 too expensive for reduced requirements

**Decision Factors:**
- **Cost**: £500-600 million per ship too expensive
- **Alternative**: Build 2-3 Type 42 destroyers for same cost
- **Quantity vs Quality**: Chose quantity (more Type 42s)
- **NATO Mission**: Area defense less critical than ASW
- **Timing**: Falklands War (1982) vindicated some decisions but questioned others

### Cancellation Impact

**Immediate:**
- Type 43 programme terminated
- Additional Type 42 destroyers ordered instead
- MESAR radar development slowed (later cancelled)
- Nuclear destroyer concept abandoned
- Design team dispersed

**Long-Term:**
- Royal Navy lacked area air defense capability 1985-2009
- Type 42 destroyers served longer than planned (to 2013)
- Type 45 programme delayed by 15+ years
- No nuclear-powered surface combatants built

**Falklands War Implications (1982):**
- Type 42 proved vulnerable (HMS Sheffield, Coventry lost)
- Type 43 might have provided better air defense
- Cancellation questioned in hindsight
- But Falklands vindicated Type 22 frigate programme

## Counterfactual Analysis

**If Type 43 Had Been Built:**

**Falklands War (1982):**
- If 1-2 Type 43s available, superior air defense
- MESAR radar and 88 Sea Dart missiles game-changing
- Might have prevented Sheffield and Coventry losses
- But programme timing meant none available 1982

**1990s-2000s:**
- 4-6 Type 43s would have provided credible area air defense
- Enabled task group operations without reliance on carriers
- Reduced reliance on Type 42 (good capabilities)
- Gap until Type 45 arrival (2009+) filled

**Budget Impact:**
- £2.5-3.5 billion programme cost
- Fewer Type 42, Type 22 frigates built
- Possible carrier sacrifice (Invincible or Ark Royal)
- Overall force structure reduced

**Type 45 Programme:**
- Type 43 success might have delayed Type 45 further
- Or Type 45 could have been Type 43 upgrade
- Nuclear option might have returned in 1990s

## Legacy

**Design Influence:**
- MESAR phased array concept reborn as SAMPSON (Type 45)
- Sea Dart Mk 2 evolution led to Sea Viper (Aster missiles)
- Large destroyer concept validated by Type 45
- Nuclear option never revisited

**Strategic Assessment:**
- Cancellation driven by economics, not capability
- Type 42 "good enough" for NATO Cold War mission
- Falklands highlighted gaps but too late to reverse
- Type 45 eventually provided similar capability 25+ years later

**Programme Management Lessons:**
- Don't gold-plate requirements beyond budget
- Quantity has a quality all its own
- Political will essential for expensive programmes
- Long gaps between classes create capability holes

**Historical Significance:**
- Last Royal Navy nuclear surface ship proposal
- End of large destroyer ambitions (until Type 45)
- Marked shift to smaller, more numerous vessels
- Symbolized post-imperial Royal Navy retrenchment

**Quote:**
"Type 43 was the destroyer the Royal Navy needed but couldn't afford - a victim of economic reality, strategic confusion, and the eternal British curse of choosing between quality and quantity. Its cancellation ensured a 25-year gap before Type 45 finally delivered the area air defense capability Type 43 would have provided in the 1990s."
