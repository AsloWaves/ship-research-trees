# Type 11 SAM (Tan-SAM Kai II)

## Overview
**Nation**: Japanese
**Type**: SAM (Short-range)
**Year**: 2011

## Specifications
- **Diameter**: 6.3"
- **Length**: 9.6 ft
- **Warhead**: 20.3 lbs (estimated)
- **Speed**: Mach 2.90
- **Range**: 10.0 nm

## System
- **Propulsion**: Solid-fuel rocket motor
- **Guidance**: Active radar (AESA) + mid-course updates
- **Target**: Aircraft, cruise missiles, UAVs
- **Platform**: Ground-based, truck-mounted (4-tube launcher)

## Notes
Successor to Type 81 SAM. Features AESA phased array seeker based on AAM-4B technology. Home-on-Jam capability. Battery: fire control vehicle + 2 launcher vehicles (8 missiles). Can operate independently via optical sight with Key-Control-Unit.
