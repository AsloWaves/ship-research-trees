## Research Prerequisites Logic

### SAM Systems Evolution
1. **US Technology Transfer** (1970-1990)
   - Tartar (2400): Initial capability → Standard MR (2401-2402)
   - Sea Sparrow (2420): Point defense → ESSM progression

2. **Indigenous Development** (2003-2024)
   - Type 03 Chū-SAM Kai (2403-2406): Medium-range indigenous SAM
   - Type 07 VLS (2410-2412): Vertical launch indigenous system
   - Progressive AESA radar integration

3. **Supplementary Systems** (1995-2008)
   - Type 91/93 adapted MANPADS for close-range defense

### SSM Systems Evolution
1. **Indigenous Anti-Ship** (1988-2023)
   - SSM-1 Type 88 (2440): First indigenous anti-ship → Type 90/12 progression
   - Type 12 family: Modern long-range anti-ship missiles
   - ASM family: Air-launched anti-ship variants (ASM-1/2/3)

2. **US Technology** (1982-2005)
   - Harpoon variants for interoperability

### ASW Systems Evolution
1. **US-Based Systems** (1977)
   - ASROC (2460): Initial ASW rocket capability

2. **Indigenous ASW** (2007-2024)
   - Type 07 VLA (2461-2463): Vertical-launch ASROC replacement
   - Integration with Type 12 torpedo family

**Starting Technologies** (1970):
- RIM-24 Tartar (Node 2400): US-supplied SAM → indigenous development
- RIM-7 Sea Sparrow (Node 2420): Point defense → ESSM
- SSM-1 Type 88 (Node 2440): First indigenous anti-ship → Type 12 evolution
- RUR-5 ASROC (Node 2460): ASW capability → Type 07 VLA

**Technology Convergence Points**:
- Node 2402 (SM-1MR): US technology mature → enables Type 03 and Type 07 branches
- Node 2443 (Type 12 SSM): Culminates indigenous anti-ship development
- Node 2422 (ESSM): NATO interoperability with indigenous systems

**High-Cost Nodes** (Research Cost >8000):
- Type 03 Mod 5 AESA (8000): Advanced AESA radar integration
- Type 03 Mod 6 (9000): Latest area defense
- Type 07 Mod 3 AESA (8500): VLS AESA upgrade
- Type 12 SSM Block 1 (8500): Extended-range anti-ship
- Type 12 SSM Block 2 (9500): Most advanced indigenous anti-ship
- ASM-3A (8000): Supersonic ramjet anti-ship

**Indigenous Technology Philosophy**:
- Early reliance on US systems (1970-1990)
- Gradual indigenous development (1988-2003)
- Full indigenous capability by 2003
- Maintain US interoperability while developing independent systems
- Focus on AESA radar integration from 2017+

**Tree Depth**: 10 generations (1970-2025, average 5.5 years/generation)

**Unique Characteristics**:
- Strong emphasis on indigenous missile development (Type 03, 07, 12, ASM series)
- Parallel US and indigenous systems for redundancy
- Advanced electronics and AESA integration
- Focus on anti-ship missiles due to maritime geography
- Late adoption of VLS systems but rapid indigenous development
