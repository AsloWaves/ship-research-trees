# Support Modules

> Support Modules are flexible equipment that enhances ship capabilities. They fill generic "Support Slots" and can be mixed based on player preference and ship role.

## Design Philosophy

Support modules allow players to customize their ships for specific roles:
- **Scouts**: Stack detection modules (radar, lookouts, floatplanes)
- **Flagships**: Maximize communications (voice radio, data links)
- **Brawlers**: Focus on damage control (DC center, firefighting)
- **ASW Hunters**: Sonar arrays, ASW weapons
- **Electronic Warriors**: RWR, jammers, chaff

## Fog of War Integration

Support modules directly affect the Fog of War system:

| Category | Effect |
|----------|--------|
| **Detection** | Expands YOUR visibility circle |
| **Communications** | SHARES visibility with allies |
| **Electronic Warfare** | DISRUPTS enemy visibility |

## Slot Rules

- Support slots accept ANY support module
- Some modules require specific Bridge tier for full functionality
- Duplicate modules may have diminishing returns
- Module weight affects ship performance
- Power-hungry modules (radar, jammers) require adequate engine output

## Era Strategy

| Era | Focus |
|-----|-------|
| **Pre-Radio (1890-1905)** | Lookouts, flags, searchlights only |
| **Dreadnought (1906-1918)** | Add hydrophones, wireless telegraph |
| **Interwar (1919-1935)** | Rangefinders, floatplanes, voice radio |
| **WWII (1936-1945)** | **Radar + voice radio = dominant** |
| **Cold War (1946-1970)** | EW critical, sonar for ASW |
| **Modern (1970+)** | Data links, automation, missiles |
