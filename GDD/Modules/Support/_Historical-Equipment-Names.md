# Historical Equipment Naming Conventions

> Equipment modules should use authentic historical designations where possible. This adds educational value and immersion while helping players learn real naval history.

## Philosophy

**Bridges** use descriptive names (Era + Type + Nation) because bridges weren't historically numbered.

**Equipment** uses historical designations because:
1. Players learn real systems (educational value)
2. Authentic feel (immersion)
3. Easy to research (Wikipedia, NavWeaps)
4. Clear national identity

### Kriegsmarine (German)

German radars used "FuMO" (Funkmess-Ortung = radio direction finding) for search and "FuMB" for detection/warning:

| Prefix | Meaning |
|--------|---------|

| FuMB | Passive radar detector (RWR) |

| FuMO 27 | 1943-1945 | Late war set | Late destroyers |
| FuMO 61 | 1944-1945 | "Hohentwiel" for U-boats | Type VII, IX, XXI |

**Radar Warning Receivers:**

## Implementation Guidelines

### Module Naming Format

```
[National Designation] [Function Description]
```

**Examples:**
- "SG Surface Search Radar"
- "Type 271 Surface Search Radar"
- "FuMO 21 Seetakt Radar"
- "Type 22 Surface Radar"
- "Mark 37 Fire Control System"

### ID Format

```
[Category]-[Nation]-[Number]
```

**Examples:**
- RAD-US-001 (SC Radar)
- RAD-UK-001 (Type 271)
- RAD-DE-001 (FuMO 21)
- RAD-JP-001 (Type 21)
- SON-US-001 (QC Sonar)
- FC-US-001 (Mark 37)

### Era Validation

Each module should validate that:
1. Radar designation matches era
2. Ship class could mount that equipment
3. National restrictions apply (can't put FuMO on US ship without capture/lend-lease)

## Educational Notes

### Why These Names Matter

Players who learn "SG Radar" in the game can:
- Look up the real system on Wikipedia
- Understand naval history discussions
- Recognize equipment in documentaries
- Appreciate the technology progression

### Historical Context

**US Radar Development:**
The US pioneered air search (SC) and surface search (SG) radar that revolutionized naval warfare. By 1944, US ships had better radar than any other navy.

**British Radar Development:**
The British invented the cavity magnetron (Type 271) which enabled centimetric radar - a war-winning technology they shared with the US.

**German Radar:**
Germany's FuMO radars were technically capable but the Kriegsmarine failed to fully integrate them. Their FuMB radar detectors became critical for U-boat survival.

**Japanese Radar:**
Japan's radar development lagged significantly. Their Type 21/22 radars were inferior to Allied equivalents, contributing to defeats at Philippine Sea and Leyte Gulf.
