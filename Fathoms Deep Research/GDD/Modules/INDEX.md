## Overview

This database contains all ship modules available in Fathoms Deep. Modules are equipment items that can be installed on ships to provide capabilities, improve performance, or enable new gameplay options.

For the module system mechanics (installation, efficiency, crew requirements), see [[04-Ship-Customization/Module-System|Module System]].
