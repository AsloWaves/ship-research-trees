## Research Prerequisites Logic

### SAM Systems Evolution
1. **Early Generation** (1956-1970)
   - Three parallel starting trees: Terrier (2000), Talos (2010), Tartar (2015)
   - Terrier → Standard MR evolution
   - Talos → heavy long-range role
   - Tartar → light destroyer role

2. **Mid Generation** (1970-1990)
   - Standard MR (RIM-66) and Standard ER (RIM-67) convergence
   - Sea Sparrow point defense line starts
   - ARM (Anti-Radiation Missile) variants branch

3. **Late Generation** (1990-2010)
   - SM-2 improvements across MR/ER types
   - ESSM replaces Sea Sparrow
   - SM-3 ballistic missile defense branch

4. **Modern Generation** (2010-2025)
   - SM-6 dual-role missiles
   - ERAM extended range variants
   - SM-3 Block II/III hypersonic intercept

### SSM Systems Evolution
1. **Early Cruise** (1955-1960)
   - Regulus I/II nuclear strike missiles

2. **Anti-Ship Era** (1977-2010)
   - Harpoon Block progression
   - Penguin helicopter-launched
   - Tomahawk land-attack cruise missiles

3. **Modern Precision** (2010-2025)
   - LRASM stealth anti-ship
   - Tomahawk Block IV/V networked strike
   - Hypersonic prompt strike systems

### ASW Systems Evolution
1. **Early ASW** (1951-1965)
   - Weapon Alpha rockets
   - ASROC development
   - SUBROC submarine-launched

2. **Vertical Launch** (1993-2020)
   - VL-ASROC replaces launcher-based systems
   - Improved sensors and homing

### Point Defense Evolution
1. **CIWS Missiles** (1980-2025)
   - Sea Sparrow adapted for CIWS
   - RAM (Rolling Airframe Missile) development
   - ESSM quad-pack cells

## Nation Progression Logic

**Starting Technologies** (1956):
- RIM-2 Terrier (Node 2000): First operational SAM → unlocks Standards
- RIM-8 Talos (Node 2010): Heavy SAM → unlocks nuclear variants
- RIM-24 Tartar (Node 2015): Light SAM → unlocks destroyer systems
- RGM-6 Regulus I (Node 2040): First cruise missile → unlocks Harpoon/Tomahawk
- RUR-4 Weapon Alpha (Node 2061): First ASW rocket → unlocks ASROC
- RIM-116A RAM (Node 2071): Point defense → unlocks ESSM

**Technology Convergence Points**:
- Node 2020 (Standard MR Block II): Unifies Tartar and Terrier lines
- Node 2005 (Standard MR): Enables advanced ER variants
- Node 2025 (ERAM): Enables both SM-3 and SM-6 branches
- Node 2058 (Tomahawk Block VA): Culminates cruise missile development
- Node 2060 (LRASM Block II): Most advanced anti-ship system

**High-Cost Nodes** (Research Cost >10000):
- RIM-161A SM-3 Block IA (12000): Ballistic missile defense
- RIM-174A Standard ERAM (10000): Extended range/altitude
- RGM-109J Tomahawk Block IV (12000): Networked strike
- AGM-158C LRASM (10000): Stealth anti-ship
- CPS Hypersonic (15000-16500): Next-generation strike

**Tree Depth**: 13 generations (1956-2025, average 5.4 years/generation)
