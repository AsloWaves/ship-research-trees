## Research Prerequisites Logic

### Fighter Evolution
1. **WWII Prop Fighters** (1940-1945)
   - F4F Wildcat (3000) → F6F Hellcat → F8F Bearcat

2. **Early Jets** (1949-1960)
   - F9F Panther/Cougar → F11F Tiger → F8U Crusader

3. **Supersonic Era** (1962-1991)
   - F-4 Phantom II variants → F-14 Tomcat lineage

4. **Modern Multi-Role** (1983-2025)
   - F/A-18 Hornet family → F-35C Lightning II

### Strike/Attack Evolution
1. **WWII Bombers** (1940-1946)
   - SBD Dauntless → SB2C Helldiver → AD Skyraider

2. **Jet Attack** (1956-1979)
   - A-4 Skyhawk family
   - A-6 Intruder series (heavy attack)
   - A-7 Corsair II (light attack)

3. **Transition to Multi-Role** (1983+)
   - F/A-18 replaces dedicated attack aircraft

### ASW Evolution
1. **Propeller ASW** (1942-1962)
   - TBF Avenger → S-2 Tracker

2. **Jet ASW** (1974-present)
   - S-3 Viking carrier-based
   - P-3 Orion/P-8 Poseidon land-based patrol

### Support Evolution
1. **AEW** (1958-2014)
   - E-1 Tracer → E-2 Hawkeye generations

2. **Electronic Warfare** (1965-2009)
   - EA-6 Prowler lineage → EA-18G Growler

3. **Reconnaissance** (1962-1996)
   - RA-5C Vigilante → TARPS pods

### Helicopter Evolution
1. **ASW Helicopters** (1961-2020)
   - SH-3 Sea King → SH-60 Seahawk family → MH-60R

2. **Modern UAVs** (2021)
   - MQ-25 Stingray unmanned tanker

## Nation Progression Logic

**Starting Technologies** (1940):
- F4F Wildcat (Node 3000): WWII fighter → jet age
- SBD Dauntless (Node 3060): Dive bomber → attack aircraft
- TBF Avenger (Node 3080): Torpedo bomber → ASW
- E-1 Tracer (Node 3090): Early AEW → Hawkeye
- SH-3 Sea King (Node 3110): ASW helicopter → Seahawk

**Technology Convergence Points**:
- Node 3040 (F/A-18): Unifies fighter/attack into multi-role
- Node 3088 (P-8 Poseidon): Modern ASW platform
- Node 3094 (E-2D): Advanced AEW with BMD capability

**High-Cost Nodes** (Research Cost >15000):
- F-35C Lightning II (20000): 5th generation stealth
- F-35C Block 4 (22000): Advanced sensor fusion

**Tree Depth**: 15+ generations (1940-2025, average 5.7 years/generation)
