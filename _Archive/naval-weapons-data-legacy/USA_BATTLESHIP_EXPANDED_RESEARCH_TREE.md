### How Modernizations Work in Gameplay

#### Option 1: Refit Existing Ships (Cheaper)
```
Player has: USS Tennessee (base version)
Player researches: Tennessee 1943 Rebuild

OPTIONS:
A) Refit USS Tennessee
   - Cost: 50% of original build cost
   - Time: 18-24 months
   - Result: USS Tennessee (Rebuilt) - all new stats

B) Build new ship in rebuilt configuration
   - Cost: 100% of new build cost
   - Time: Full build time
   - Result: New ship with rebuilt stats
```

#### Option 2: Sidegrades (Different Specializations)
```
Example: New York 1926 Refit
- Base New York: Balanced battleship
- 1926 Refit: +40% AA, +9000nm range, -2 knots speed

Not strictly better, but specialized for different roles
```
