## Japanese Type Designation System

**Type XX = Imperial Year**
- Formula: Type Number = (Year - 1868)
- Example: Type 94 = 1868 + 94 = **1932** (design year for Yamato guns)
- Example: Type 3 = 1868 + 3 = **1871** (Taisho 3 = 1914)
- Example: Type 88 = 1868 + 88 = **1956** (error - actually Showa 3 = 1928)

**Note**: Some guns use Taisho (1912-1926) or Showa (1926-1989) eras:
- Type 3 (Taisho 3) = 1914
- Type 10 (Taisho 10) = 1921
- Type 88 (Showa 3) = 1928
- Type 91 (Showa 6) = 1931
- Type 94 (Showa 9) = 1934
- Type 96 (Showa 11) = 1936
- Type 98 (Showa 13) = 1938

- 100 guns √ó 4 types = 400 base turrets

### Tactical Variants (1,700 total)
Applied to base turrets to reach 2,100 total:
- **LA** (Light Armor) - Faster, lighter protection
- **HA** (Heavy Armor) - Enhanced protection, slower
- **AA** (Anti-Aircraft) - High-angle capable
- **Surf** (Surface) - Surface combat optimized
- **Shld** (Shielded) - Gun shield protection
- **Enc** (Enclosed) - Fully enclosed mount
- **CD** (Coastal Defense) - Land-based variant

### Japanese Turret Characteristics
- Emphasis on **high muzzle velocity** over shell weight
- Advanced optical rangefinding systems
- Compact turret designs due to space constraints
- Slower traverse rates compared to Western designs
- Heavy armor on capital ship turrets (410-650mm face on Yamato)

### Turret Distribution
- **Battleship turrets** (46cm-36cm): Heavy armor, slow traverse, high firepower
- **Cruiser turrets** (20.3cm-15cm): Balanced speed and protection
- **Destroyer turrets** (12.7cm-10cm): Fast traverse, light armor
- **AA turrets** (8cm-25mm): High elevation, rapid traverse

## Notable Japanese Naval Features

### Design Philosophy
1. **Quality Over Quantity**: Fewer but more powerful ships
2. **High Velocity**: Prioritized muzzle velocity over shell weight
3. **Optical Excellence**: World-class rangefinding systems
4. **Torpedo Emphasis**: Heavy torpedo armament on all warships
5. **Night Combat**: Specialized night combat doctrine and training

### Unique Technologies
1. **Sanshiki AA Shells**: Anti-aircraft shrapnel shells (Type 3)
2. **Long Lance Torpedoes**: Type 93 oxygen torpedoes, most powerful in WWII
3. **Pagoda Masts**: Distinctive tall superstructures for observation
4. **Large Cruisers**: "Heavy cruisers" that exceeded treaty limits (*Mogami* class)
5. **Super-Battleships**: Yamato class - largest battleships ever built

### Caliber Preferences
- **Battleships**: 46cm (Yamato), 41cm (Nagato), 36cm (Kongo)
- **Heavy Cruisers**: 20.3cm standard (8")
- **Light Cruisers**: 15.5cm (Mogami), 14cm, 15cm variants
- **Destroyers**: 12.7cm (5") standard
- **AA**: 25mm Type 96 most common, 10cm/65 Type 98 for AA destroyers
