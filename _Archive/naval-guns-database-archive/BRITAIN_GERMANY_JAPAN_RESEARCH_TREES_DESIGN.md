# Britain, Germany, Japan Research Trees - Design Document

**Date**: January 2025 (Continuation Session)
**Purpose**: Design comprehensive research trees for Britain, Germany, Japan (Cruisers, Destroyers, Submarines)
**Status**: DESIGN PHASE

---

## Design Philosophy

Following USA research tree pattern:
- FREE starting ships (2 per type)
- Multiple tech branches (Main Line, Alternative, Specialist, Modernization)
- Tier progression: 1 (Starting) → 9-10 (Modern)
- Historical accuracy with gameplay balance
- Special abilities reflecting national doctrines

## BRITAIN RESEARCH TREES

### British Cruisers (2600-2699) - ~45 nodes

**Tech Branches:**
1. **Starting** (2600-2601): Pelorus-class (1896) FREE, Leander-class (1931) FREE
2. **Protected/Armored Main Line** (2602-2607): Cressy → Drake → Warrior → Duke of Edinburgh → Devonshire → Monmouth
3. **Light Cruiser Main Line** (2608-2620): Town (1910) → Arethusa → C-class → D-class → E-class → Emerald → Leander (1931) → Amphion → Arethusa (1934) → Town (1936) → Fiji → Minotaur
4. **AA Specialist Branch** (2621-2623): Dido → Bellona → Scylla
5. **Heavy Cruiser Branch** (2624-2629): Hawkins → County (Kent) → County (London) → County (Norfolk) → York → Exeter
6. **Post-War/Guided Missile** (2630-2640): Tiger → Blake → County (DDG) → Bristol → Sheffield → Type 22 → Type 23
7. **Modernization** (2641-2643): Tiger (Modernized), County (Modernized), Sheffield (Modernized)
8. **Paper Ships** (2644-2645): Design CA, Future Cruiser Concepts

**National Characteristics:**
- Emphasis on balanced designs and seakeeping
- Strong AA capabilities (Dido-class)
- Long-range commerce protection
- Special Ability Ideas: "Empire Reach" (+15% range), "Royal Navy Standards" (+10% reliability)

### British Destroyers (2700-2799) - ~42 nodes

**Tech Branches:**
1. **Starting** (2700-2701): 27-knotter (1894) FREE, River-class (1903) FREE
2. **Pre-WWI** (2702-2705): Tribal (F) → Acorn (H) → Beagle (G) → Acasta (K)
3. **WWI Main Line** (2706-2710): Admiralty M → R-class → V-class → W-class → Modified W
4. **Interwar Standard** (2711-2717): A → B → Tribal (1936) → J → K → N → O/P
5. **WWII Emergency** (2718-2722): Q/R → S/T → U/V → W/Z → Battle-class
6. **Post-War** (2723-2729): Daring → County (DDG) → Type 42 Batch 1 → Type 42 Batch 2 → Type 42 Batch 3
7. **Modern** (2730-2735): Type 45 Batch 1 → Type 45 Batch 2 → Type 83 (Future)
8. **Alternative** (2736-2738): Hunt (escort) → Black Swan → River (frigate)
9. **Modernization** (2739-2741): Various FRAM-equivalent upgrades

**National Characteristics:**
- Alphabetical class naming system
- Strong ASW focus (ASDIC pioneers)
- Escort and convoy protection emphasis
- Special Ability Ideas: "ASDIC Mastery" (+20% ASW), "Atlantic Veteran" (+10% endurance)

## Research Cost Scaling

**Formula**: `Base_Cost × Tier_Multiplier × Rarity_Multiplier × Nation_Modifier`

**Tier Multipliers**:
- Tier 1 (FREE): 0 RP
- Tier 2: 5,000-10,000 RP
- Tier 3: 10,000-20,000 RP
- Tier 4: 15,000-25,000 RP
- Tier 5: 20,000-30,000 RP
- Tier 6: 25,000-40,000 RP
- Tier 7: 35,000-50,000 RP
- Tier 8: 45,000-65,000 RP
- Tier 9: 60,000-80,000 RP
- Tier 10: 75,000-100,000 RP

**Rarity Multipliers**:
- Common (mass production): 1.0×
- Uncommon: 1.2×
- Rare (limited production): 1.5×
- Very Rare (experimental): 1.8×
- Unique (1-3 ships): 2.0×
- Paper Ships: 1.5× (high RP but -10% reliability penalty)

## Build Cost & Time Scaling

**Build Cost by Era**:
- Pre-Dreadnought (1890-1905): $500K-$5M
- Dreadnought Era (1906-1920): $2M-$10M
- Interwar/Treaty (1921-1939): $5M-$20M
- WWII (1940-1945): $10M-$100M
- Early Cold War (1946-1960): $50M-$300M
- Modern (1961-2000): $200M-$2B
- Contemporary (2001+): $1B-$4B+

**Build Time**:
- Small vessels (DD, SS): 12-24 months
- Medium (CL, SSN): 24-48 months
- Large (CA, CG, SSBN): 48-84 months
- Experimental/Complex: +20-50% time
- Mass Production: -10-20% time

## Special Abilities by Nation

**Britain**:
- "Empire Reach": +15% operational range
- "Royal Navy Standards": +10% reliability, +5% crew efficiency
- "ASDIC Pioneer": +20% ASW effectiveness

- "Handelskrieg": +20% commerce raiding effectiveness
- "Diesel Innovation": +15% range, -10% maintenance
- "Wolfpack Tactics": +25% coordinated submarine attacks
- "High Pressure Steam": +10% speed for destroyers
- "Type XXI Revolution": +30% submerged speed (Type XXI)

**Japan**:
- "Long Lance": +30% torpedo range, +25% damage
- "Night Battle Doctrine": +25% night combat effectiveness
- "Special Type": +20% torpedo capability, +10% speed
- "I-400 Unique": Carries 3× Seiran aircraft (UNIQUE ability)
- "Oxygen Torpedoes": +35% torpedo damage, +50% range
